# 聊吧论坛前端页面规范

## 一、配色方案

对于页面的所有颜色，都应该使用

```ts	
export interface ThemeVariables {
  '--primary': string;
  '--secondary-warm-1': string;
  '--secondary-warm-2': string;
  '--secondary-cool': string;
  '--neutral-bg': string;
  '--neutral-module': string;
  '--neutral-text-main': string;
  '--neutral-text-secondary': string;
  '--neutral-border': string;
  '--neutral-shadow': string;
}
```

这些颜色进行配色，由`ThemeConfig.ts`和`ThemeUtil.ts`来动态修改颜色，达到切换主题的效果，如文字使用

`p{color:var(--neutral-text-main);}`

目前支持主题`teal`,`pink`,`yellow`,`dark`，通过`ThemeUtil`来加载
